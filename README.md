# 얌얌 — 날씨 기반 맛집 추천 서비스

> 원 프로젝트: 2022.07 – 2022.12 (교내 팀 프로젝트 2인 · Python, Pandas, NumPy, REST API)
> 계정 재생성 과정에서 원본 코드가 유실되어, 핵심 로직을 재구현한 저장소입니다.

## 지표가 알려준 문제
학교 주변 맛집 추천으로 시작 → 1차 테스트 만족도 2.8점(5점 만점), "흔하다"는 피드백.
기능 부족이 아니라 차별성 부족이 원인이라고 진단.

## 가설과 검증
"날씨가 음식 선택에 영향을 준다" — 설문 50여 명으로 검증 후
네이버 검색 API + 공공데이터(날씨)를 연동한 추천 로직 설계.
담당: 데이터 수집·전처리, 추천 로직 설계·성능 검증.

## 결과
동일 참가자 재테스트에서 만족도 4.2점(+1.4). "독특해서 좋다"는 반응으로 가설 증명.
